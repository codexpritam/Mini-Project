# Expense Tracker for Personal and Group Expense Management

## Project Description

Expense Tracker is a full-stack web application developed using the MERN stack that helps users manage both personal and group expenses on a single platform.

Many existing applications focus either on personal expense tracking or group expense management separately. This project solves that problem by combining both functionalities into one application, making expense management more convenient and efficient.

Users can track their daily expenses, manage group expenses, monitor spending, and maintain transaction records through a clean and responsive user interface.

## Features and Functionalities Implemented

### Personal Expense Management
- Add personal income and expenses
- Categorize transactions
- View expense history
- Track overall spending

### Group Expense Management
- Create groups
- Join groups using invite codes
- Add group expenses
- Split expenses among group members
- View group transaction history

### Additional Features
- Responsive user interface
- REST API integration
- Secure database storage using MongoDB
- Real-time expense updates
- User-friendly dashboard
